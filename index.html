About BLAST Lab

BLAST Lab은 경계 없는 인증(Borderless Authentication)과 지능형 기기 보안(Secure Smart Things) 분야의 원천 기술 연구를 통해 현재와 미래 보안 문제를 해결하는 것을 목표로 합니다.

BLAST Lab aims to solve current and future security challenges through fundamental research in Borderless Authentication and Secure Smart Things.

Borderless Authentication

인증 대상 · 적용 기술 · 적용 도메인의 경계를 넘나드는 인증 기술 연구

Research on authentication technologies that transcend the boundaries of authentication targets, applied techniques, and application domains.

Secure Smart Things

자율주행 자동차, 로봇, 스마트홈 디바이스 등 다양한 사물 환경에서의 보안 연구

Security research across diverse smart environments including autonomous vehicles, robots, and smart home devices.

Network Security

네트워크 프로토콜, 네트워크 보안 아키텍처, 네트워크 침입 탐지 및 방어 메커니즘 연구

Research on network protocols, network security architectures, and network intrusion detection and defense mechanisms.
